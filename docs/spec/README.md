# Спецификации системы Tomorrow Bot

Оригинальные файлы спецификации находятся в директории `workai/` в корне проекта.

## Содержимое workai/

Директория `workai/` содержит исходные технические задания, исследования
и спецификации, разработанные командой. Эти файлы являются источником
истины при проектировании архитектуры системы.

## Использование

При разработке новых модулей всегда сверяйтесь со спецификациями в `workai/`
для обеспечения соответствия архитектурным решениям и бизнес-требованиям.
